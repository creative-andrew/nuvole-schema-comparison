Main Conclusion

The site is in a good technical position after the schema cleanup: sampled pages now expose one Yoast JSON-LD graph, useful citation and user-triggered agents can fetch public content, the sitemap is reachable, and /llms.txt is already present.

The main gap is strategic, not mechanical: robots.txt still has no explicit AI crawler policy.
